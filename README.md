# watanabe-farm-demo

## structure

```text
watanabe-farm/
├─ index.html              ← トップページ（LP本体）  
├─ about.html              ← 農園紹介ページ（任意）  
├─ crops.html              ← 作物紹介ページ（任意）  
├─ contact.html            ← お問い合わせページ（任意）  
│
├─ /assets/                ← 素材フォルダ（全てここに整理）  
│  ├─ /images/             ← 写真やイラスト類  
│  │   ├─ top-hero.jpg  
│  │   ├─ farm-view.jpg  
│  │   ├─ rice.jpg  
│  │   ├─ cherry.jpg  
│  │   └─ logo.png  
│  │
│  ├─ /css/  
│  │   └─ style.css        ← 全体のスタイル  
│  │
│  ├─ /js/  
│  │   └─ main.js          ← スクロール演出やメニュー開閉など  
│  │
│  └─ /fonts/              ← Webフォントを使う場合  
│
├─ README.md               ← プロジェクト説明（GitHub用）  
├─ .gitignore              ← 不要ファイルをGitに含めない設定  
└─ LICENSE                 ← ライセンス（公開時に任意で設定）
